Aide en ligne

Vous trouverez ici des informations utiles à la bonne utilisation de cette application.

Si vous avez besoin de contacter un administrateur, vous pouvez utiliser le formulaire de contact en allant dans Autres puis Contact.

Détail des domaines

Chaque domaine répertorie les informations les plus utiles et importantes. Vous y trouverez des conseils, informations et coordonnées des différentes activités.

Par exemple : Comment se rendre à Paris ?

Je clique sur la page Transport pour voir tous les moyens de transport qui me sont proposés.

Si je choisis d'y aller par train, j'ai toutes les informations sur les gares les plus proches et je peux me rendre sur le sîte internet afin de réserver mes billets.

Recherche

La fonction recherche vous permet de rechercher un domaine ou une activité comportant ce mot.

Compte

Cette page vous permet de voir ou modifier vos informations personnelles.

Vous pouvez aussi consultez votre avancement dans les tâches que vous avez commencées.

Autres

Dans ce sous-menu, vous pourrez découvrir l'aide en ligne ou encore les convictions de l'association.
